# PartyVibes

Sitio estático para PartyVibes — catering y organización de fiestas infantiles.

Descripción
------------
Este repositorio contiene el HTML, CSS y JS del sitio estático de PartyVibes.

Cómo contribuir / publicar
-------------------------
1. Abre tu terminal en la carpeta del proyecto:

```bash
cd /g/Proyecto/PartyVibes
```

2. Verifica cambios, añade y commitea:

```bash
git status
git add .
git commit -m "Add .gitignore and README"
```

3. Si ya tienes el remote creado (origin), empuja los cambios:

```bash
# si tu rama principal se llama 'main'
# git push -u origin main

# si tu rama principal se llama 'master'
git push -u origin master
```

Publicar con GitHub Pages
-------------------------
- Ve al repositorio en GitHub → Settings → Pages.
- En "Source" selecciona branch `main` o `master` y la carpeta `/ (root)`.
- Guarda; en unos minutos GitHub publicará el sitio en `https://USERNAME.github.io/REPO`.

Notas
-----
- Asegúrate de no subir credenciales ni archivos sensibles (usa `.env` y agrégalo a `.gitignore`).
- Si necesitas que cree también un `LICENSE` o una `CNAME`, dime cuál prefieres.
